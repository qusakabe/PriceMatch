# Документация агрегатора продуктовых сервисов PriceMatch

## Общая информация о проекте
**Название проекта:** агрегатор доставки продуктов "PriceMatch"

**Цель проекта:** Разработка веб-платформы для сравнения предложений различных сервисов доставки продуктов по цене, времени доставки и наличию акций.

**Ключевые функциональные возможности:**
- Поиск и фильтрация товаров по категориям.
- Отображение цен и условий доставки из различных сервисов.
- Персонализированные списки покупок.
- Интеграция с картами лояльности для отображения персонализированных скидок.

## Выбранные источники данных
Проект агрегирует данные из следующих сервисов:
- **ВкусВилл** – сервис доставки натуральных продуктов с широким ассортиментом.
- **Азбука Вкуса** – премиальный сервис доставки, ориентированный на качественные продукты.
- **Самокат** – один из лидеров экспресс-доставки продуктов, обеспечивающий быстрый сервис.

**Критерии выбора сервисов:**
- Популярность среди пользователей.
- Доступность API или возможность парсинга.
- Широкий ассортимент товаров.
- Возможность оформления заказа онлайн.
- Регулярное обновление информации о ценах.

## Структура базы данных

**User**
- id (PK)
- firstname
- lastname
- phoneNumber
- password

**Profile**
- id (PK)
- user (FK OneToOne)
- saleCards (FK RN)
- favorite (FK ManyToMany)
- lists (FK RN)

**SaleCard**
- id (PK)
- amount
- profile (FK)
- service (FK)

**List**
- id (PK)
- name
- profile (FK)
- products (FK ManyToMany)

**Product**
- id (PK)
- service (FK)
- name
- image
- price
- url

**Service**
- id (PK)
- name
- productsCount

## Алгоритм работы системы
1. Пользователь вводит название товара в поиске.
2. Система отправляет запрос в базу данных и возвращает список сервисов, предлагающих данный товар.
3. Фильтрация данных по цене и срокам доставки.
4. Отображение информации в виде списка.
5. Пользователь выбирает нужное предложение и переходит на сайт сервиса доставки для оформления заказа.

## Заключение
Проект "PriceMatch" призван упростить процесс выбора выгодных предложений на рынке доставки продуктов, сэкономить время пользователей и повысить прозрачность ценовой политики сервисов доставки. Благодаря регулярному обновлению данных, персонализированным рекомендациям и расширенным возможностям фильтрации, пользователи смогут получать наиболее актуальную и полезную информацию, что делает платформу незаменимым инструментом для всех, кто регулярно заказывает продукты онлайн.
